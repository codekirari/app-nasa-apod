**app-nasa-apod** is a front-end application designed to query the **official NASA API (APOD)** to display the astronomical image and corresponding explanation for the current date or a user-selected date.

This project serves as a practical demonstration of several key front-end skills, including **consuming and processing data from an external API**, **manipulating the DOM**, and creating a clean user interface to present dynamic, external information.

* **API Consumption:** Executes GET requests to the NASA APOD API endpoint.
